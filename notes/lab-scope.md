# Lab Scope

## Purpose

This lab simulates a basic Security Operations Center environment for learning log collection, alert detection, and incident investigation.

## Environment

The lab is built using virtual machines in a controlled local environment.

## In Scope

- Wazuh server setup
- Windows endpoint monitoring
- Kali Linux test activity
- Failed login detection
- Basic scanning detection
- Suspicious command investigation
- SOC-style documentation

## Out of Scope

- Public targets
- Third-party networks
- Real production systems
- Malware deployment
- Credential theft
- Unauthorized access attempts

## Rules of Engagement

- Only test systems owned and controlled in the lab
- Do not attack public IP addresses
- Do not collect real user credentials
- Do not upload sensitive logs
- Use screenshots and logs only from the controlled environment
